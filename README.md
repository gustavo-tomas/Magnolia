## Requirements
<!-- Add links -->
- C++20
- Gcc 11.4.0
- Python 3.10.12
- Lua 5.4
- Cmake 3.22.1
- Vulkan SDK 1.3.268

## Build
To build and run the program use

```
python3 build.py debug
```
